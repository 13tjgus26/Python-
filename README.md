# [프로젝트 제안서] 동네 핫플 분포 시뮬레이터
## 프로젝트 이름: 트렌드 기반 동네 핫플 밀집도 분석기
2차원 리스트 좌표계를 활용하여 핫플레이스의 위치 데이터를 텍스트 지도로 시각화하고, 특정 구역의 밀집도를 분석하는 프로그램입니다.
### 풀고싶은 문제
현재 인스타그램에서 유행하는 카페나 맛집 정보는 많이 소개되고 있지만, 막상 그 동네에 갔을 때 이 카페랑 맛집이 서로 얼마나 멀리 있는지, 어느 골목에 핫플이 몰려 있는지 등 한눈에 파악하기가 어렵습니다. 저는 위치정보시스템학 전공자로서 지도의 원리를 이용해, 내가 가고 싶은 장소들을 격자 좌표 위에 직접 표시해보고 싶었습니다. 제가 배운 파이썬의 기본 기능(리스트, 반복문)만으로 우리 동네의 핫플 지도를 제 손으로 직접 그려보는 경험을 하기 위해 이 프로젝트를 기획하였습니다. 단순히 목록을 나열하는 것보다 지도로 시각화하면 어디가 제일 번화한 곳인지 더 쉽게 이해할 수 있기 때문입니다.
### 핵심 기능 3가지
1. 좌표 기반 장소 등록: 사용자로부터 장소명, 카테고리 (맛집, 카페 등), X/Y 좌표값을 입력받아 객체(리스트)에 저장하는 기능

2. 텍스트 지도 시각화: 2차원 리스트를 활용하여 터미널 창에 * 또는 카페, 식당 등을 기호로 장소의 분포 현황을 출력하는 기능
3. 구역별 밀집도 계산: 특정 좌표 범위를 입력하면 해당 구역 내에 포함된 핫플레이스의 개수와 가장 많이 나타나는 카테고리를 분석해주는 기능
### 예상 사용자
특정 동네의 상권 분포를 데이터를 확인하고 싶은 사람들 또는 나 자신
### 결과물 형태
CLI (Command Line Interface) - 파이썬 터미널 창에서 텍스트로 동작
### 사용할 Python 개념
• 변수: 지도 크기, 장소 이름, 좌표값 저장

• 리스트 (2차원 리스트): 격자 형태의 지도 공간 구현 및 장소 데이터 목록 관리

• 반복문 (for): 지도를 화면에 한 줄씩 출력하고, 리스트 내의 장소 데이터를 순회하며 검색

• 조건문 (if): 입력된 좌표가 지도 범위 내에 있는지 확인하고, 카테고리에 따라 다른 기호 표시

• 함수: 지도 생성, 장소 추가, 지도 출력, 밀도 분석 등 기능별 코드 모듈화

• 문자열: 장소 정보 출력 및 텍스트 기반 지도 기호 렌더링
